# ReviewForge AI 🚀

AI-powered pull request review platform for detecting security issues, bugs, and performance problems automatically.

## ✨ Features
- AI PR reviews
- Security detection
- Performance analysis
- Bug detection
- Modern dashboard UI
- GitHub integration ready

## 🛠 Tech Stack
- React
- TypeScript
- Vite
- GitHub API
- Replit

## 🚀 Run Locally

```bash
git clone https://github.com/jithin310-dj/reviewforge-ai.git
cd reviewforge-ai
npm install
npm run dev
```
